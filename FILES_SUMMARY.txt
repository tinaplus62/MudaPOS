JAVASCRIPT UTILITIES
─────────────────────────────────────────────────────────────────────────────
📄 js/common.js (400+ lines)
   - Shared utility functions used across pages
   - Format functions: formatDate, formatCurrency, formatNumber
   - Modal controls: openModal, closeModal
   - Sidebar controls: toggleSidebar, closeSidebar
   - Validation helpers: validateEmail, validatePhone, validateRequired
   - HTML security: escapeHtml for XSS prevention
   - Utility functions: debounce, deepClone, addDelegatedEventListener
   - Page initialization: initPage, setActiveMenuItem
   
   USAGE: <script src="js/common.js"></script>
   
   BENEFITS:
   ✅ Eliminates repeated code across pages
   ✅ Consistent function behavior
   ✅ Easy updates (change once, works everywhere)
   ✅ Better maintainability

📄 js/auth-utils.js (200+ lines)
   - Secure authentication utilities
   - Password hashing with salt: hashPassword, verifyPassword
   - Secure session management: storeSecureSession, getSecureSession
   - Session cleanup: clearSecureSession, secureLogout
   - Input sanitization: sanitizeInput
   - Safe event handlers: createSafeEventHandler
   
   USAGE: <script src="js/auth-utils.js"></script>
   
   FEATURES:
   ✅ SHA-256 hashing with salt
   ✅ Session timeout (24 hours)
   ✅ XSS prevention with sanitization
   ✅ Secure logout clearing all data
   ✅ Password verification helpers

📄 js/notifications.js (350+ lines)
   - Modern notification system replacing 200+ alert() calls
   - Toast notifications: showSuccess, showError, showWarning, showInfo
   - Modal dialogs: alert, confirm, prompt
   - Auto-dismiss toast messages
   - Responsive design for mobile
   - Styled to match application theme
   
   USAGE: <script src="js/notifications.js"></script>
   
   REPLACES:
   ❌ alert() → showSuccess(), showError(), etc.
   ❌ confirm() → await confirm() with options
   ❌ prompt() → await prompt() with validation
   
   BENEFITS:
   ✅ Professional UX (no blocking alerts)
   ✅ Mobile-friendly notifications
   ✅ Consistent styling
   ✅ Async/await support
   ✅ Customizable options

📄 js/validation-utils.js (350+ lines)
   - Comprehensive form & input validation
   - Field validators: isValidEmail, isValidPhone, isValidNumber, etc.
   - Form validation: validateForm, validateField
   - Error display: displayValidationErrors, markInvalidFields
   - Custom validation rules support
   
   USAGE: <script src="js/validation-utils.js"></script>
   
   FEATURES:
   ✅ Email, phone, number validation
   ✅ Date, code, bank account validation
   ✅ Custom validation rules
   ✅ Real-time error feedback
   ✅ Field highlighting for errors

📄 js/export-utils.js (300+ lines)
   - CSV export and data handling
   - Export to CSV: exportToCSV, exportTableToCSV
   - Transaction export: exportTransactions
   - Print functionality: printHTML
   - Copy to clipboard: copyTableToClipboard
   
   USAGE: <script src="js/export-utils.js"></script>
   
   FEATURES:
   ✅ CSV export with formatting
   ✅ Table to CSV conversion
   ✅ Automatic timestamp in filename
   ✅ Print with styling
   ✅ Copy to clipboard

📄 js/firebase-config-safe.js (50 lines)
   - Safe Firebase configuration loading from environment variables
   - Loads API keys from .env file instead of hardcoding
   - Validation for missing configuration
   - Exports all necessary Firebase modules
   
   USAGE: import { dbCloud, auth, ... } from './js/firebase-config-safe.js'
   
   REPLACES: ./js/firebase-config.js (deprecated)
   
   BENEFITS:
   ✅ API keys not visible in source code
   ✅ Different keys per environment (dev/staging/prod)
   ✅ Easier credential rotation
   ✅ Better security practices
